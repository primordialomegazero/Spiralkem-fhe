# Contributing to Spiralkem-FHE

## Quick Start

```bash
git clone https://github.com/primordialomegazero/Spiralkem-fhe.git
cd Spiralkem-fhe
```

## Pull Request Guidelines

- All tests must pass (`make test`)
- New features need test coverage
- Follow the φ-framework philosophy
- No breaking changes to existing API

## Code Style

- C11 standard
- Follow OpenSSL naming conventions
- φ-constants must use `1.6180339887498948482`

## Issues

Found a bug? Open an issue with:
- liboqs version
- Compiler version
- Steps to reproduce
- Expected vs actual behavior
